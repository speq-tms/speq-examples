# Canonical fixtures

This repository contains two canonical fixtures:

- `in-repo-mode/.speq/...` for service repo embedding.
- `test-repo-mode/...` for dedicated test repository layout.
- `test-repo-mode-jsonplaceholder/...` for public API smoke validation with suite-per-entity grouping.

These fixtures are used by `speq-cli validate` tests to guarantee consistent mode discovery and schema validation.
